# Character Bible Builder

Interactive web form for creating AI persona definitions.

## Features

- Step-by-step wizard interface
- Covers all aspects of persona creation:
  - Identity & background
  - Voice & communication style
  - Knowledge domains
  - Personality traits
  - Guardrails & boundaries
  - Improvisation zones
- Generates downloadable markdown file
- IPAI branded

## Usage

Fill out each section, then download your completed Character Bible as a markdown file.
